## 本周词汇目标

掌握首批 10 个教育类表达，重点练习：

- `tuition-free university education`
- `gain admission to university`
- `students from low-income families`
- `provide financial assistance`
- `lifelong learning`
- `eligibility criteria`

复习采用第 1、3、7、14 天间隔，并以“看中文能说出英文、能够正确造句”为掌握标准。
